# AgentDesk Product Requirements

This document captures the full product roadmap for AgentDesk. Phase 1 implements only the runnable foundation: monorepo structure, FastAPI health endpoint, Next.js landing page, dashboard shell, sidebar navigation, Docker Compose, environment template, and startup documentation.

Future phases will add database models, knowledge base ingestion, tickets, LangGraph agent workflows, human approvals, tracing, and evaluations.
